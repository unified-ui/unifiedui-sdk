# Proofs of Concept (POCs)

This directory contains proof-of-concept implementations and explorations for new SDK features.

## Structure

```
pocs/
├── README.md           # This file
└── ...                 # POC scripts organized by feature
```

## Guidelines

- Each POC should be self-contained
- Add a brief description at the top of each script
- POCs are **not** production code and are excluded from linting rules
