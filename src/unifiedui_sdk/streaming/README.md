# unifiedui_sdk.streaming

Standardized streaming response protocol for unified-ui integrations.

## Status

**Planned** — This module is not yet implemented. See the streaming concept design document in the repository root for the planned architecture.

## Planned Scope

- SSE event models matching the agent-service streaming protocol
- Stream builder/writer utilities for custom agent backends
- Support for reasoning tokens, tool call events, and structured streaming
- Framework adapters for LangChain/LangGraph streaming
